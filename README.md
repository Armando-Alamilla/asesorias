#### _Recordatorio de comandos para correr el proyecto_

```
$ export COMPOSE_FILE=local.yml

$ docker-compose build
$ docker-compose up

$ docker-compose ps

$ docker rm -f asesorias_django_1
$ docker-compose run --rm --service-ports django

$ docker-compose down
```
